# Screenshots

Drop your app screenshots here. Exact file names expected:

| File | What to capture |
|---|---|
| `dashboard.png` | `/app` → logged in → Dashboard screen |
| `pets.png` | `/app` → My Pets screen showing pet cards |
| `care.png` | `/app` → Care screen showing medications table |
| `appointments.png` | `/app` → Appointments screen |
| `login.png` | `/app` → Login screen |
| `hero.png` | `/` → Landing page hero section |

## How to take them

1. Open http://localhost:3001/app in Chrome
2. Press F12 → click the device toolbar icon (phone/tablet icon)
3. Set to "iPhone 14 Pro" — 390x844
4. Navigate to each screen
5. Right-click → Save image as → save to this folder with the exact name above

Then run: `git add . && git commit -m "assets: add real app screenshots" && git push`
